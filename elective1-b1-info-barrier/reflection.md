# B1 任务复盘

> 戴昊杰 (DEEP007) | 2026.07.15

## 1. 这次任务让我发现了哪些信息差？

**最大发现：Agent 框架已经从"功能对比"进化到"团队栈匹配"阶段。** 2025 年大家还在问"LangChain vs CrewAI 哪个好"，2026 年 6 月的 LangChain 官方文章给出了完全不同的答案——"取决于你的云服务商、语言偏好和现有工具链"。这不是简单的信息更新，是思维方式的变化：选择不是基于"谁更强"，而是基于"谁和你的基础设施匹配"。

**第二发现：MCP 生态的"信号噪声比"问题比我预想的严重。** Task 003 测试运行中我用了 PulseMCP 的数据（17,000+ Server），但这次搜索中没有一个来源能告诉我"哪些 Server 是生产级可靠的"。这说明 MCP 生态还处于野蛮生长阶段——信息筛选能力本身就是壁垒。

## 2. 我之前是否存在只看二手总结、短视频或社群消息的问题？

**部分存在。** Task 001/002 的信息来源有我主动搜的原始资料（LangChain 文档、CrewAI GitHub），但也有依赖 Claude 帮我"搜索+总结"的部分——这是二手总结，但不是短视频/社群那种低质量二手。

做完 B1 后我意识到：Firecrawl + 直接打开原始来源阅读 > AI 总结 > 社群二手信息。AI 总结的问题是"去掉了作者的语气和判断"——而恰恰是语气和判断（比如 LangChain 文章坦诚承认"抽象层在调试时会有摩擦"）才是信息中最有价值的部分。

## 3. 我是否学会了区分真实机会和短期热点？

**框架层面——学会了。** 现在我能清楚地判断：MCP/A2A 协议标准比任何单一框架更重要（学协议不学框架），Claude Code 的选择是因为它匹配我的工具栈而非因为它最热门。

**Server 层面——还需要工具。** 17,000+ MCP Server 中选 10 个可靠的——我需要一套"Server 质量筛选标准"（GitHub 活跃度、issue 响应速度、下载量趋势、社区反馈）。B1 任务让我意识到这是下一步要建立的系统。

## 4. 我最有价值的一条信息是什么？为什么？

**"46% 高级开发者最爱 Claude Code，但生产力溢价不均匀"（来自 uvik.net 数据）。**

最"有价值"不是因为它最权威——而是因为它同时告诉我：Claude Code 确实强（用的人最多且满意度最高），但也不是万能药（不同任务类型的效率提升差异大）。

这种"既好又不完美"的信息比纯正面或纯负面的信息更有价值——因为它能指导我合理预期：Claude Code 在文件组织/网页生成/README 生成方面很强，在复杂调试方面可能需要更多人工干预。

## 5. 哪条信息看起来很热，但我认为暂时不适合自己？

**Microsoft Agent Framework（MAF）1.0 GA。** 

理由很清楚：MAF 的 GitHub 10K 星、AutoGen+Semantic Kernel 统一、graph-based workflows——这些确实听起来很厉害。但它对 Azure AI Foundry 的深度依赖意味着：我不在这个生态里，MAF 的大部分优势对我不可用。而社区反馈也显示非 Azure 场景的 provider adapter 有 bug。

**不做"看起来很热但对自己没用"的投入——这是 B1 任务训练的核心能力。**

## 6. 我接下来会如何建立长期 AI 信息雷达？

三管齐下：

1. **被动源（降低信息获取成本）：** 关注 LangChain Blog、Anthropic 官方博客、MCP GitHub org——这些是 Agent 和 MCP 领域的一手信息源。
2. **主动搜索（每周 15 分钟）：** 用 Firecrawl 搜 "AI agent framework 2026" "MCP server" "Claude Code" 看每周有没有新东西。
3. **实践验证（最关键的过滤层）：** 新工具必须实际跑一次才判断价值——Task 003 测试运行就是这种方式的示范。不空评。

## 7. 这个任务如何帮助我和普通 AI 使用者拉开差距？

普通使用者看 AI 信息 = 刷短视频/公众号 → "AI 又出新东西了，我好焦虑" → 什么都不做，继续焦虑。

我做 B1 后 = 确定一个主题 → 找 5 个高质量来源 → 分析每条信息对我的实际价值 → 标注哪些热门但我不追 → 写出具体行动建议。

**差距不在"我知道得更多"，在"我知道什么对我有用、什么对我没用"——这是信息素养，不是信息量。** 普通使用者的问题不是信息太少，是缺少信息过滤器。B1 任务训练的就是这个过滤器。
