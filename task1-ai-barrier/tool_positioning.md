# AI 工具定位表 — 戴昊杰

## 我已接触/使用的 AI 工具

| 工具名称 | 主要用途 | 适合任务 | 我现在是否需要 | 我准备如何使用 | 可能的问题 |
|----------|----------|----------|---------------|---------------|-----------|
| **Claude Code** | AI Agent 编程助手，终端原生，支持 Skills/Hooks/MCP 编排 | 项目开发、自动化脚本、Web 应用搭建、数据处理、多文件重构 | ✅ 核心工具 | 作为日常主力 Agent，驱动所有项目从设计到部署的完整闭环。持续优化 CLAUDE.md 和 Skills 配置 | 上下文窗口限制；模型选择（Opus vs Sonnet）需要按任务分配合适等级，避免浪费；MCP Server 较多时需注意响应速度 |
| **ChatGPT** | 通用对话、思维整理、文案生成、学习辅助 | 任务拆解、复盘写作、学习规划、头脑风暴 | ✅ 辅助工具 | 用于任务规划和复盘，不适合代码密集型任务。与 Claude Code 互补：ChatGPT 做思路发散，Claude Code 做落地执行 | 输出偏泛化，需要人工判断和具体化；不适合深度代码工作 |
| **Gemini** | 多模态理解、长文本分析、资料检索 | 图片理解、长文本摘要、论文阅读、多轮检索 | 🟡 可尝试 | 用于辅助理解视觉内容和分析大篇幅资料，探索 Gemini 的多模态能力在项目中的应用 | 输出质量不稳定，中文能力弱于 Claude 和 ChatGPT；部分场景下推理深度不足 |
| **Codex (OpenAI)** | AI 编程 Agent，与 Claude Code 同方向的竞品 | 代码生成、项目辅助、Agent 操作 | 🟡 有使用 | 作为 Claude Code 的补充和对比，重点观察两者在同一个任务上的表现差异。可用来做 Agent 间的交叉验证 | 功能和 Claude Code 高度重叠，同时维护两个 Agent 体系成本高 |
| **GitHub Copilot** | IDE 内嵌代码补全 | 行级/函数级代码补全、样板代码生成 | 🟡 辅助 | 在 VS Code 中作为轻量级代码补全工具，不用于复杂任务驱动 | 补全质量不稳定，与 Claude Code 的 Agent 模式相比粒度太细，效率提升有限 |
| **Firecrawl** | Web 搜索+抓取+数据提取一体化 API | 网页数据采集、竞品监控、AI 信息聚合、结构化数据提取 | ✅ 核心工具 | 通过 MCP Server 集成到 Claude Code，实现"AI 自动搜索→抓取→分析→报告"全链路 | API 调用有成本；某些 JS 渲染页面可能需要 waitFor 参数调优 |

## 我计划学习的工具

| 工具名称 | 主要用途 | 适合任务 | 我现在是否需要 | 我准备如何使用 | 可能的问题 |
|----------|----------|----------|---------------|---------------|-----------|
| **CrewAI** | Multi-Agent 编排框架，定义 Agent 角色、任务和协作流程 | 多步骤复杂任务分解、Agent 间协作、自动化工作流 | 🟡 短期计划 | 在 DEEP 营任务三（Agent 雏形搭建）中探索使用，设计一个简单的 2-3 Agent 协作 demo | 学习曲线较陡，框架本身快速迭代中；简单任务上 Multi-Agent 可能过度工程化 |
| **MetaGPT** | Multi-Agent 框架，以软件公司 SOP 方式编排 Agent（产品经理→架构师→工程师→测试） | 复杂软件项目、团队协作模拟 | 🟡 短期计划 | 理解其 SOP 式协作设计理念，思考如何应用到自己的项目工作流中 | 比 CrewAI 更重，启动成本高；需要较强的工程基础 |
| **Gradio / Streamlit** | AI Demo 和 Web UI 快速搭建 | AI 模型的在线演示、数据分析结果展示、项目作品展示 | ✅ 短期需要 | 将数据分析项目和 Agent 工具包装成可在线访问的 Demo 页面，形成可视化的作品展示 | 和 Claude Code 的集成方式需要探索；复杂 UI 受限 |

## 我的工具策略

- **主引擎**：Claude Code（驱动所有项目级工作）
- **信息输入**：Firecrawl MCP（Web 数据获取）
- **浏览器操作**：Playwright MCP / Browser-Act（自动化交互）
- **辅助思考**：ChatGPT（规划、复盘、发散）
- **代码轻补全**：GitHub Copilot（IDE 内快速补全）
- **未来扩展**：CrewAI / MetaGPT（多 Agent 编排）、Gradio（作品展示）
