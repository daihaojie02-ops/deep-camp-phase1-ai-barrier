## 使用 Claude Code 的关键经验

1. **需求描述要具体：** 第一次 Prompt 包含了"暗黑极客风格"、"不需要交互 JS"这些细节——越具体 Claude Code 输出的第一版越接近目标。
2. **验证后迭代：** 第一版出来不是终点——浏览器打开后发现数据不对，再让 Claude Code 改。两轮迭代就达到了目标。
3. **单文件策略：** 因为是展示页面不是应用，用单文件 HTML（内嵌 CSS/JS）就够了——不需要复杂的项目结构。
