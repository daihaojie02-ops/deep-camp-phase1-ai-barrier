## 一、我为什么要学习 Python

在参加 DEEP 营之前，我认为 Python 是程序员的工具，与我无关。但做完 Task 001-003 后，我发现一个瓶颈：Claude Code 可以帮我分析和写文档，但如果我想让 AI 帮我"做事情"（而不是"写东西"），中间需要一层执行能力——这层就是 Python。

Python 不是"编程语言"，是"AI 的执行层"——把我和 AI 的共识变成可运行的结果。没有它，AI 永远停留在聊天窗口里。

## 九、报错或修改记录

**主要报错：** Unicode 编码错误——Windows GBK 终端不支持 emoji 字符（"✅"）。

**修复方案：** 把 emoji 替换为纯 ASCII 文本（"✅" → "[DONE]"）。

**学到的：** 跨平台开发的编码问题——以后写代码尽量避免在终端输出 emoji。

**数据个性化修改：** 把 Claude Code 生成的示例任务名换成了我的真实 DEEP 营任务——让输出有真实价值而非演示用途。
