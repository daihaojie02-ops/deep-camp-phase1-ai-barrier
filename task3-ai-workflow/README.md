## 核心方法论

```
TechScout Agent = 标准作业程序(SOP) + 3 Prompt模板 + 输出模板 + 质量检查表 + 复盘迭代

不是代码自动化 Agent——是方法论自动化 Agent。
下次遇到任何新技术领域：
  填主题 → 搜(Step1) → 拆(Step2) → 判(Step3) → 比(Step4) → 查(Step5) → 出(Step6)
  60分钟从"一无所知"到"有理由的决策"
```
